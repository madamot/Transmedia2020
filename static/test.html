An SVG that shrinks into the Bootstrap navbar

Made possible by:

  • skrollr by Alexander Prinzhorn
  • A point in the right direction by Matt Dempsey (Thanks man!)

How it was done:

This was stuck before the closing </body>

      <script src="js/skrollr.min.js"></script>
      <script type="text/javascript">
          var s = skrollr.init();
      </script>

And the the element to be animated had it's HTML5 data- attributes added like so:

      <div id="logo" data-0="width:153px;left:50vw;margin-left:-77px;top:30vh;position:fixed;" data-800="width:51px;left:50vw;margin-left:-25px;top:0;" data-top="position:fixed;">

Easy peasy!
